> A structured collection of JavaScript programs covering everything from the basics to advanced ES6+ concepts.

### 📁 Project Structure

```
javascript-programming-learning/
├── JavaScript Part-01/     # Basics — variables, loops, arrays, objects
├── JavaScript Part-02/     # Hoisting, closures, lexical scope
└── DOM Tutorial/           # DOM manipulation with HTML & CSS
```

### 🧠 Concepts Covered

| Module | Topics |
|---|---|
| `JavaScript Part-01` | Variables, data types, loops, arrays, objects, functions |
| `JavaScript Part-02` | Hoisting, closures, execution context, scope chain |
| `DOM Tutorial` | DOM selection, events, dynamic styling |

### 🚀 Getting Started

```bash
git clone https://github.com/isinghabhishek/javascript-programming-learning.git

# Open any .js file in your editor or run with Node.js
node "JavaScript Part-01/01file.js"

# For DOM Tutorial, open index.html in a browser
```

### ✅ CI — Lint & Validate

Every push runs ESLint on all `.js` files and html-validate on `.html` files via GitHub Actions.
